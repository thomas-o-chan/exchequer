# Exchequer

Exchequer is a simple app for displaying chess.com API data in a simple wiki format. 

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project is also deployed on https://exchequer-gamma.vercel.app/

## Testing

This repo contains a test suite run suing vitest with a few simple tests for demonstrative purposes. There is not significant coverage.
